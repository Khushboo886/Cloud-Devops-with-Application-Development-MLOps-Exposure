# WHAT IS pm2?
-> Production grade node.js process manager that makes it easy to manage and scale your node.js applications.

# WHY WE USE pm2?
-> Can manage your applications with ease, restart them automatically in the case that they crash.\
-> Can deploy them to multiple servers with zero downtime.\
-> Provies a built in load balancer.

# HOW TO SET UP pm2?
-> To install pm2 command globally run this command in your cmd: npm install pm2 -g
